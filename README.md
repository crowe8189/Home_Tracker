# Crowe's Nest Build – Home Construction Manager

Local-first Streamlit app for your $350k new home at 450 SR 27, Whitwell, TN 37397.

**Run locally**:
1. `pip install -r requirements.txt`
2. Rename `.streamlit/secrets.toml.example` → `.streamlit/secrets.toml` and add your Gemini key
3. `streamlit run app.py`

Fully functional: Dashboard, Budget, Gantt Roadmap, Receipts+OCR, AI Assistant (Gemini), Smart Checklist, Permits, Alerts.

Future improvements in the Settings page notes.
